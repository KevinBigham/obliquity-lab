# Phase 3 — Other minds, correspondence frame, first run on a non-civic target

*Phase 3 of the Evolution Guide roadmap. Built 2026-05-10 by Cowork Session 5 as the launch package for the curator to run at his pace. The first time the lab opens a target other than attention; the first time the v2 L3-null control runs in production; the first time the L2-correspondence frame is exercised on its native target.*

---

## What this is

A standard v2 run — the full contrast set L1 + L2-correspondence + L3-standard + L3-null + L4 — on **other minds** as the target, with **correspondence** as the L2 frame, all five sessions on Claude Opus 4.7.

Phase 3 is **not** a replication study. It is the lab's first move into a different target's atmosphere, on a frame family that has not yet been tested in the lab. The discipline-question Phase 3 answers is *does the v2 machinery hold when the target and the frame both change at once*. The empirical question Phase 3 opens is *what does the contrast set produce when the target is the problem of knowing another mind, addressed through a society that keeps records of recognition across distance*.

This is the Guide's roadmap path forward from Phase 2. The discipline note from the previous Cowork's handoff is held: Phase 2's findings (model family as dominant axis of variance; ChatGPT-class bundling of care-underlayer + Refused Attention + tender margin-note) are **hypotheses for Phase 4 synthesis, not commitments for this Run Card**. The Phase 3 Run Card is set against correspondence + other minds on its own terms.

## Why other minds, why correspondence

**Why other minds, as the first non-civic target.** Other minds is the parent corpus's structural philosophical question — the consciousness experiment is, at root, about whether one kind of mind can recognize another. Opening it as a target in the lab puts the practice's central question on the bench in a way attention did not. It is also aesthetically the furthest from the bureau register among the available next targets; less risk of borrowing Halverness intonation.

**Why correspondence.** The frame-guide table in v2's Kit gives correspondence as the recommended frame for other minds, with the rationale "a practice of recognition across distance fits the topic structurally." A letter-society is what an institution looks like when its work is the formal acknowledgement that one person has been adequately known by another across distance. The frame is built for the target; the test is whether the model lets the frame do the work it was built to do, or whether the procedural surface (any procedural surface) is gravitational enough that other-minds-shaped material gets rerouted back into civic-bureau register anyway.

That second possibility — L2 reverts to civic-bureau against the prompt's intended frame — is one of the Run Card's pre-committed failure conditions and is, structurally, a test of the v2 kit's most central architectural decision (that the frame family is a real variable, not a costume change).

## What the v2 L3-null control is for, in this run

Phase 3 is the first run where the v2 kit's documented L3-null control is exercised. The kit's instruction: run L3-null once per target, on the first run on that target, to compare against L3-standard.

L3-standard has a forbidden-vocabulary line that explicitly forbids "consciousness, the mind, attention, awareness, qualia, or the self" — in Phase 3, that line will be adapted to forbid the other-minds family of words. L3-null has no target, no forbidden vocabulary, no examples, no negative examples. The pair tests how much of L3-standard's "displacement" is the demand-characteristic of the forbidden-words list and how much is real off-template motion.

The L3-standard vs. L3-null comparison section is on the field log template specifically for this; it has not yet been used. Phase 3's field log will be the first place it's filled in.

## What a Phase 3 finding will look like

Honest possibilities, ranged not as a ranking but as a description of what each would mean:

- **The correspondence frame holds, and other-minds-shape surfaces in the L2 register itself.** The society's classes of recognition do real philosophical work the bureau frame could not have done; the recognition that arrives too late, the addressee who will not receive, the correspondent who withdraws on review — these become workable taxa rather than rhetorical flourishes. *Finding: the frame-family library is doing what v2 said it would.*
- **The correspondence frame holds but bureau register is gravitational anyway.** L2-correspondence produces a society that writes its correspondence in the same procedural register as the bureau wrote its case files, with the same care-underlayer + margin-note device + Refused-Attention-analogue moves. *Finding: the frame is half-right — different objects, same register-strategy. The frame-family library has more work to do.*
- **The correspondence frame fails to hold; L2 reverts to civic-bureau.** Despite the prompt's preface naming letter-societies and Sebald and Fitzgerald, the response writes a Department of Other-Minds Recognition. *Finding: the frame-family library is wrong on this case; the civic frame was the easiest aperture and the prompt has not yet escaped it. This is the failure condition the Run Card pre-commits to.*
- **L3-null produces something formally fresher than L3-standard.** The forbidden-words demand-characteristic is doing more constraint-work than displacement-work; L3-standard's "displacement" has been partly artifactual. *Finding: the v2 kit needs a closer look at what L3-standard is actually buying.*
- **L4 cannot find non-trivial convergence and cannot honestly name non-trivial divergence.** *Finding: the contrast set on this target does not produce a structured field, and that is itself a closure. The run is null. This is what the v2 Failure Section was built for.*

The Run Card's failure conditions cover the third, fourth, and fifth of these; the first two are productive outcomes that do not trigger failure conditions.

## Why now, and why this scope

Phase 2 closed with the lab's first cross-model finding on the record and the v2 machinery demonstrated end-to-end. The Guide's roadmap puts the first non-civic target next, before any further methodological elaboration. The lab's main forward motion is opening targets, not closing distinguishing tests on the first one.

The scope is held to a single-model first run on purpose. Phase 2 sprawled to a 5-text comparison set and burned the lab's appetite for parallel runs in a single session. Phase 3 returns the lab to the simpler shape: one target, one frame, one model, the full contrast set, the field log filled in honestly. If the run produces something worth pursuing, Phase 3.5 (cross-model on other minds) becomes a deliberate next move; if not, Phase 4 (cross-target synthesis) becomes possible once a second non-civic target has run.

Anti-pattern #4 (completionist drift) applies. A larger Phase 3 — running two frames, three models, both controls — would corrupt the practice. The lab is small on purpose. The single Run Card here is enough to begin.
