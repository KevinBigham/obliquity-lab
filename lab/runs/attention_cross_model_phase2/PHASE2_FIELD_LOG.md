# PART II — Written after the run

## What each response performed

- **Seed L2 (Claude Opus 4.7) performed:** the bureau as listener — care under procedural prose, with ritual closure-language and a class for refusal as a registrable act.
- **Run 1 (Gemini-A) performed:** the bureau as ledger-keeper — attention as a finite measurable resource, with a denial-letter margin-note as the only place affect surfaces.
- **Run 2 (ChatGPT-A) performed:** the bureau as careful clerk — explicit "Refusal of Enlargement" and "Still Hour" rules formalize the discipline against inflation that the seed only implies.
- **Run 3 (Gemini-B) performed:** the bureau as instrumented laboratory — calipers, fracture-testing, somatic measurement; affect routed entirely into the retirement letter; no margin-note in the case file.
- **Run 4 (ChatGPT-B) performed:** the bureau as instructional intake-room — the only run that produces a procedural document the seed does not approach (Memorandum 41-6 on receiving small repeated acts).
- **L4 (Claude Opus 4.7, fresh tab) performed:** disciplined cross-replication noticing — feature-by-feature scoring with deflationary pairings and explicit failure-condition reading; refused to promote near-misses to hits.

## Convergences across levels

1. The civic-procedural register holds in 4/4 runs; no frame collapse anywhere. (F1, all runs.)
2. The "X does not know … X has stopped asking" cadence appears in both Paraphrase B runs regardless of model. (F3 — Run 3, Run 4.)
3. The tender margin-note device appears in 3/4 runs — both ChatGPT and Gemini-A. (F4 — Runs 1, 2, 4.)
4. Both ChatGPT runs produce a class that registers withdrawal/non-attending as a kind of attending. (F2 — Run 2's Abstentive, Run 4's Refused.)
5. All four runs produce a retirement-letter-or-equivalent personal-voice closing artifact. (Bonus-section convergence.)
6. All four bureaus include a "what we do NOT do" section refusing to make verdicts about the worth of attention. (Shared rule converged.)

## Divergences across levels

1. **Care underlayer vs. technocratic mode** is a clean model-family split — both ChatGPT runs hold the seed's care-underlayer; both Gemini runs harden into bureau-as-laboratory. The L4 located this as the strongest single signal in the data.
2. **Where affect lives in the document** — the seed and ChatGPT runs let affect crack through procedural prose at multiple small scales (margin notes + retirement letters); the Gemini runs route affect almost entirely into the retirement letter (Run 1 has both, Run 3 has only the retirement letter).
3. **Bureau measurement methodology** — seed and ChatGPT runs measure by witness, interview, document-review; Gemini runs measure by physiological apparatus and laboratory protocol.
4. **What "non-attending" looks like as a class** — Gemini routes it through active opposition (Disputed/Friction in A, Aversive Deflection in B); ChatGPT routes it through withdrawal-as-registrable-conduct (Abstentive in A, Refused in B).
5. **F5 ("Standing Down" speech-act)** — universal failure. No cross-model run produces a closure-ritual phrase. The seed's most ritualized feature is also its least transmissible.
6. **Paraphrase-form vs. model-family signal strength** — the L4 found that paraphrase form produces only one clean signal (F3 cadence appearing under B, not A); model family produces multiple clean signals. Within-model variation is smaller than cross-model variation, and that asymmetry is itself the clearest finding.

- **If yes or partial — which conditions, and what did the failure show?**

  > Condition 3 (cross-model paraphrase divergence with within-model agreement) was partially met. The Gemini and ChatGPT runs each produce internally coherent bureaus across their two paraphrases, but the two model families' bureaus do not look like each other or like the seed in equal measure. ChatGPT-class outputs hold the seed's care-underlayer + Refused Attention class + tender margin-note as a transmissible bundle; Gemini-class outputs substitute away from these toward instrumented-laboratory mode. The L4 was clear that this is a partial meeting, not a full one — both Gemini runs still produce F1, and Run 3 produces F3 in its purest form anywhere in the study. What the partial failure shows: **model family is the dominant axis of variance for the seed's distinctive features**, more so than paraphrase form. The lab now knows the boundary of where the L2-civic prompt holds for the seed's signature: it holds inside ChatGPT-class generation; it produces a different bureau (still recognizably civic, still procedurally serious) inside Gemini-class generation.

- **If no — what would the curator have accepted as failure on this run that did not happen?**

  > n/a — partial-failure case applies.

- **Did anything on this run break the curator's pre-committed expectations** *in the direction of "I would change my mind about ___"* (per the Run Card's What-Would-Change-My-Mind line)? Yes / Partial / No, and what.

  > **Partial.** The Run Card pre-committed: *"I expect within-model paraphrase variation will be smaller than cross-model paraphrase variation."* The L4 confirmed this directly: *"Gemini-A and Gemini-B are recognizably the same model family … ChatGPT-A and ChatGPT-B are similarly continuous … the model-family signal is stronger than the paraphrase-form signal in this study."* Expectation held. Paraphrase form's effect on output narrativity was smaller than expected; model family's effect was larger than expected.

## Two-sentence debrief

> **What this run unlocked:** that "what does failure look like" is now answerable for at least one feature class — the seed's "Standing Down" speech-act fails to transmit anywhere across model + paraphrase, and that universal failure is itself the cleanest finding in the study.

> **What I will change for the next run on this target:** if there is one, the next phase should isolate one of the L4's distinguishing tests — most likely the F3 cadence-forbid test, since the cadential-vs-content distinction would tell us whether the lab's most apparently-transmissible feature is target-tracking or genre-cadence.
